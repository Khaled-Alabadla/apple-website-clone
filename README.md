# Apple Website Clone

This project is a high-fidelity clone of various sections of the Apple website, including the homepage, iPhone product page, accessories, and support portal.

## 🚀 Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Modern Aesthetics**: Built with premium CSS transitions, gradients, and typography.
- **Dark Mode Support**: Seamless toggle between light and dark modes.
- **Interactive Components**: Including custom Swiper carousels and mega-menus.

## 🛠️ Built With
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Tailwind CSS
- Swiper.js (for carousels)

## 📂 Project Structure
- `index.html`: Homepage clone.
- `iphone.html`: iPhone product page.
- `accessories.html`: Shopping and accessories section.
- `support.html`: Apple Support portal.
- `/assets`: CSS, JS, and image assets.
